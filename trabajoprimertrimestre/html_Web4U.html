Etiquetas HTML

Cabeceras

Las cabeceras se definen con las etiquetas <h1> hasta <h6>.
La cabecera <h1> es la más importante y la <h6> es la menos importante.
Ejemplo:
<html>
<body>
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
</body>
</html>

Párrafos

La etiqueta <p> define un párrafo.
Ejemplo:
<html>
<body>
<p>This is a paragraph.</p>
<p>This is a paragraph.</p>
<p>This is a paragraph.</p>
</body>
</html>

Enlaces

La etiqueta <a> define un enlace.
El atributo más importante del elemento <a> es el atributo "href", que indica el destino del enlace.
El texto del enlace es la parte que será visible para el lector.
Al hacer clic en el texto del enlace, el lector será enviado a la dirección URL especificada.
Ejemplo:
<html>
<body>
<h1>HTML Links</h1>
<p><a href="https://www.w3schools.com/">Visit W3Schools.com!</a></p>
</body>
</html>

Imágenes

La etiqueta <img> se utiliza para incrustar una imagen en una página web.
Técnicamente, las imágenes no se insertan en una página web, las imágenes están vinculadas a páginas web.
La etiqueta <img> crea un espacio para la imagen. La etiqueta <img> está vacía, solo contiene atributos y no tiene etiqueta de cierre.
La etiqueta <img> tiene un atributo obligatorio "src" que especifica la ruta a la imagen.
Ejemplo:
<html>
<body>
<img src="img_chania.jpg">
</body>
</html>

Tablas

La etiqueta <table> define una tabla HTML.
Cada fila de la tabla se define con una etiqueta <tr>. Cada encabezado de tabla se define con una etiqueta <th>. Cada celda de la tabla se define con una etiqueta <td>.
De forma predeterminada, el texto de los elementos <th> está en negrita y centrado.
Ejemplo:
<html>
<body>
<table>
<tr>
<th>Firstname</th>
<th>Lastname</th>
<th>Age</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
<tr>
<td>John</td>
<td>Doe</td>
<td>80</td>
</tr>
</table>
</body>
</html>
